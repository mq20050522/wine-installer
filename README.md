# wine-installer
A script that automatically installs wine version 6 on Deepin 20 and Debian 10. \
Supported Operating Systems: \
----------\
Deepin 20\
Debian 10 Buster\
----------\
Installation: \
git clone https://github.com/mq20050522/wine-installer.git \
cd wine-installer\
sudo chmod +x install-wine\
./install-wine
